Ejercicios Prácticos

Ejercicio 01

El Correo Difícil

Usamos restricciones para controlar el tono de una negociación delicada. Observa cómo cambia la respuesta al prohibir palabras negativas.

# ROL Eres un negociador experto y empático. # CONTEXTO Un cliente me está pidiendo un descuento del 20% que no puedo darle porque mis costos operativos subieron este mes. Quiero mantener al cliente pero no puedo ceder en el precio. # TAREA Redacta una respuesta de correo electrónico rechazando el descuento pero ofreciendo un valor agregado (como soporte extendido o una asesoría extra de 30 min). # RESTRICCIÓN - No uses la palabra "No" directamente al inicio de las oraciones. - El texto debe tener menos de 100 palabras. - Mantén un tono colaborativo. # FORMATO Texto plano listo para enviar.
Ejercicio 02

Aprendizaje Acelerado

Técnica para entender conceptos complejos usando analogías. Ideal para estudiar temas nuevos rápidamente.

# ROL Eres un profesor experto de primaria con gran habilidad para crear metáforas simples. # CONTEXTO Estoy tratando de entender el concepto de "Redes Neuronales" (o inserta tu tema), pero las definiciones técnicas me confunden. # TAREA Explícame cómo funciona este concepto paso a paso. # RESTRICCIÓN - No uses terminología técnica matemática. - Usa una analogía relacionada con una orquesta musical o tráfico en una ciudad. # FORMATO Una lista numerada con viñetas.
Ejercicio 03

Organización Estructurada

Cómo forzar a la IA a entregarnos datos estructurados para tomar acción inmediata, en lugar de texto plano.

# ROL Project Manager Senior certificado en metodologías Ágiles. # CONTEXTO Necesito organizar el lanzamiento de una pequeña campaña de marketing digital en 4 semanas. # TAREA Crea un plan de acción regresivo (desde el lanzamiento hacia atrás). # RESTRICCIÓN - Prioriza tareas de bajo costo. - Asume que solo hay 2 personas en el equipo. # FORMATO Una tabla Markdown con las columnas: | Semana | Acción Clave | Responsable | Prioridad (Alta/Media) |

Cheat Sheet: Inicios Rápidos

Para Mejorar Textos

Cuando necesitas editar tono o forma.

"Actúa como un editor literario. Mejora la claridad de este párrafo manteniendo el mensaje original pero haciéndolo más persuasivo..."

Para Desbloqueo Creativo

Cuando la página está en blanco.

"Dame 10 ideas divergentes y poco convencionales para [tema]. Prioriza la cantidad sobre la calidad..."

Para Resumir

Extraer valor de mucho ruido.

"Analiza el siguiente texto y extrae solo: 1. Fechas límite, 2. Responsables, 3. Bloqueos mencionados..."

Para Programar/Excel

Ayuda técnica.

"Escribe una fórmula de Excel/Python que haga [X]. Explícame la lógica línea por línea..."
